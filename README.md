# Clinical Prediction Modeling in R

A bilingual Codex skill for reproducible clinical prediction modeling in R, including data contracts, Logistic/Cox/competing-risk workflows, validation, calibration, discrimination, clinical utility, and transparent reporting.

本仓库只包含 Codex skill 本身，不包含任何患者数据、研究结果、分析项目或教材 PDF。

## Language versions

- [简体中文版本](zh-CN/README.md)
- [English version](en-US/README.md)

Each language directory contains an installable `clinical-prediction-modeling-r` skill with its `SKILL.md`, UI metadata, references, and R validation scripts.

## Scope

The skill is designed for research-grade, reproducible clinical prediction workflows. It does not provide individual medical advice and does not claim clinical deployability without appropriate external validation, calibration, clinical utility assessment, and governance.
