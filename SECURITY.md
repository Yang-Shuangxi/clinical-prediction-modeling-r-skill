# Security and privacy / 安全与隐私

## English

This repository is a documentation-and-validation skill, not a clinical deployment service. Never commit patient-level data, identifiers, access tokens, private paths, or generated research outputs. If a secret or identifiable dataset is exposed, revoke the credential, remove the file from the repository history, and notify the maintainer privately.

## 中文

本仓库是文档和结构检查 skill，不是临床部署服务。严禁提交患者级数据、患者标识符、访问令牌、个人路径或研究结果文件。若误提交凭据或可识别数据，应立即撤销凭据、从仓库历史中清理文件，并私下通知维护者。
